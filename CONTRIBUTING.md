# Contributing to vanvlaenderen.org

Thank you for your interest in contributing!

## Getting Started

1. Fork the repository on GitHub.
2. Clone your fork:
   ```bash
   git clone https://github.com/<your-username>/vanvlaenderen.org.git
   cd vanvlaenderen.org
   pnpm install
   ```
3. Create a feature branch:
   ```bash
   git checkout -b feature/your-feature-name
   ```

## Development Workflow

- Run `pnpm dev` to start the local development server.
- Run `pnpm build` to verify the production build succeeds before opening a PR.
- Keep commits small and focused; write clear commit messages in the imperative mood.

## Pull Request Process

1. Ensure your branch is up to date with `main`.
2. Describe what your PR does and why.
3. Link related issues using `Closes #N`.

## Reporting Issues

Use [GitHub Issues](https://github.com/iamabotama/vanvlaenderen.org/issues) to report bugs or request features.
